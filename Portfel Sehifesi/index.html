Bildyim Kodlaşdırma Dilləri

  • HTML:HTML (Hipermətn İşarələmə Dili) - Veb səhifələrinin strukturunu yaratmaq üçün istifadə edilən standart bir dilidir. Bu dil, başlıqlar, paraqraflar, şəkillər və linklər kimi elementləri təsvir etmək üçün işarələrdən (etiketlərdən) istifadə edir, bu da brauzerlərin səhifəni düzgün şəkildə göstərməsinə imkan verir. HTML bir proqramlaşdırma dili deyil, vebin təməlidir və veb brauzerlərdə işləyir.
  • CSS: CSS (Cascading Style Sheets) - Veb-səhifələrin tərtibatı və xarici görünüşünü tənzimləyən bir təsvir dilidir. Bu texnologiya sayəsində veb-saytların görünüşünü (rəngləri, mətnlərini, yerləşməsini) fərdiləşdirmək, dinamik və responsiv (ekran ölçülərinə uyğun tənzimlənən) dizaynlar yaratmaq mümkündür. CSS, HTML elementlərinə təsir edən seçici və xüsusiyyətlərdən ibarət sintaksisdən istifadə edir.
  • Java Script: JavaScript – internet səhifələrin yaradılmasında geniş istifadə olunan proqramlaşdırma dilidir. JavaScript müxtəlif dilllərdən ilhamlanaraq yaradılmışdır. 1995-ci ilə Brendan Eyx tərəfindən yazılmışdır. JavaScript Obyekt Yönümlü Proqramlaşdırma (OYP) dilidir.
